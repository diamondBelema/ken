Installation, content format, and command reference.

Install

Download binary

Pre-built binaries for Linux and Windows:

PlatformDownload
Linux (x86_64) ken
Windows (x86_64) ken.exe 
# Linux
curl -LO https://github.com/diamondBelema/ken/releases/latest/download/ken
chmod +x ken && mv ken ~/.local/bin/

From source

git clone https://github.com/diamondBelema/ken.git
cd ken
go build -o ken ./cmd/ken

Requires Go 1.21+. Move the binary somewhere on your $PATH:

mv ken ~/.local/bin/

Run directly

go run ./cmd/ken

Content format

Content lives at ~/Documents/learn/subjects/<subject>/ and is always read-only to ken. Progress is stored at ~/.local/share/ken/.

~/Documents/learn/subjects/
  biochemistry/
    concepts/
      glycolysis.md
    flashcards/
      glycolysis.md
    quizzes/
      glycolysis.md

Concept file

concepts/glycolysis.md
---
format_version: 1
type: concept_set
set: Glycolysis
concepts:
  - id: c-glycolysis
    name: Glycolysis
    parent_id: null
  - id: c-pfk1
    name: Phosphofructokinase-1
    parent_id: c-glycolysis
---

## c-glycolysis
The metabolic pathway that breaks down glucose into pyruvate, yielding a net 2 ATP.

## c-pfk1
Rate-limiting enzyme of glycolysis. Inhibited by ATP and citrate.

Flashcard file

flashcards/glycolysis.md
---
format_version: 1
type: flashcard_set
set: Glycolysis Cards
cards:
  - id: bch-001
    concept_id: c-pfk1
    front: "Rate-limiting enzyme of glycolysis?"
    back: "Phosphofructokinase-1 (PFK-1)"
  - id: bch-002
    concept_id: c-glycolysis
    front: "Net ATP yield of glycolysis?"
    back: "2 ATP"
---

Quiz file

quizzes/glycolysis.md
---
format_version: 1
type: quiz_set
set: Glycolysis Quiz
questions:
  - id: bch-q001
    concept_id: c-pfk1
    type: mcq
    question: "Which enzyme is rate-limiting?"
    options:
      - "Hexokinase"
      - "PFK-1"
      - "Pyruvate kinase"
    answer: 2
    explanation: "PFK-1 is the rate-limiting enzyme."
  - id: bch-q002
    type: true_false
    question: "Glycolysis yields 4 ATP net"
    answer: false
    explanation: "Net yield is 2 ATP (4 produced - 2 invested)."
---

CLI reference

Command Description
ken Launch the dashboard
ken subjects List subjects with counts
ken flashcards <subject> Study flashcards (5-level grading)
ken quiz <subject> Take a quiz (mcq, true/false, fill-blank)
ken progress [subject] View concept confidence breakdown
ken notes <subject> View and manage notes
ken summaries <subject> View and manage summaries
ken read <subject> Read lecture content
ken lint [subject] Validate content files (exits 1 on errors)

ken lint accepts --json for machine-readable output.

Registry commands

Command Description
ken search <query> Search the package registry
ken add <author/package> Install a study package from the registry
ken list List installed packages
ken remove <author/package> Uninstall a package
ken package Generate ken.yaml manifest from local content
ken publish Publish packages to the registry (requires gh CLI)

Package format

Packages are identified as author/package (e.g. diamondBelema/nucleic-acid). Each package includes a ken.yaml manifest:

ken.yaml
id: diamondBelema/nucleic-acid
name: Nucleic Acid
version: 1.0.1
author: diamondBelema
description: "DNA and RNA structure, replication, transcription, translation"
subjects:
  - nucleic-acid
tags: [biochemistry, molecular-biology]

See the registry for available packages.

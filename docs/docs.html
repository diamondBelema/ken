Install

From source

git clone https://github.com/diamondBelema/ken.git
cd ken
go build -o ken ./cmd/ken

Requires Go 1.21+. The binary outputs to ./ken in the project root. Move it somewhere on your $PATH:

mv ken ~/.local/bin/

Run directly

go run ./cmd/ken

Content format

Content lives at ~/Documents/learn/subjects/<subject>/ and is always read-only to ken. Progress is stored separately at ~/.local/share/ken/.

~/Documents/learn/subjects/
  biochemistry/
    concepts/
      glycolysis.md
    flashcards/
      glycolysis.md
    quizzes/
      glycolysis.md
    notes/
      lecture-14.md

Minimal concept file

concepts/glycolysis.md
---
format_version: 1
type: concept_set
set: Glycolysis
concepts:
  - id: c-glycolysis
    name: Glycolysis
    parent_id: null
  - id: c-pfk1
    name: Phosphofructokinase-1
    parent_id: c-glycolysis
---

## c-glycolysis
The metabolic pathway that breaks down glucose into pyruvate, yielding a net 2 ATP.

## c-glycolysis:summary
First step of cellular respiration. Net 2 ATP per glucose. Occurs in cytoplasm.

## c-pfk1
Rate-limiting enzyme of glycolysis. Inhibited by ATP and citrate.

## c-pfk1:summary
Commits glucose to glycolysis. Inhibited by ATP/citrate, activated by AMP/F-2,6-BP.

set: Glycolysis Quiz
questions:
  - id: bch-q001
    concept_id: c-pfk1
    type: mcq
    question: "Which enzyme is rate-limiting?"
    options:
      - "Hexokinase"
      - "PFK-1"
      - "Pyruvate kinase"
    answer: 2
    explanation: "PFK-1 is the rate-limiting enzyme."
  - id: bch-q002
    type: true_false
    question: "Glycolysis yields 4 ATP net"
    answer: false
    explanation: "Net yield is 2 ATP (4 produced - 2 invested)."
---

CLI reference

Command Description
ken Launch the dashboard (default command)
ken subjects List subjects with concept/card/quiz counts
ken flashcards <subject> Study flashcards for a subject (5-level grading)
ken quiz <subject> Take a quiz for a subject (mcq, true/false, fill-blank)
ken progress [subject] View concept-level confidence breakdown
ken stats View detailed stats and confidence trends
ken notes <subject> View and manage notes for a subject
ken summaries <subject> View and manage summaries for a subject
ken read <subject> Read lecture notes and content for a subject
ken lint [subject] Validate content files and report issues (exits 1 on errors)

ken lint accepts --json for machine-readable output — useful for agent pipelines that need to self-check before handing off content.

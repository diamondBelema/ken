How it works

Three layers of learning

Drop Markdown files into a folder. ken builds a three-layer learning system based on cognitive science. Content stays read-only.

  1. Familiarize with ken map. Walk the concept tree, read summaries, mark concepts as seen. No pressure — just exposure to the material.

  2. Reflect with ken reflect. Type your own explanation before seeing the canonical answer. Self-explanation strengthens retention more than re-reading.

  3. Master with ken flashcards & ken quiz. Active recall under pressure. Bayesian confidence with time-decay tracks what you actually know.

Features

Built for focused learning

Three-layer system

Familiarity → Reflection → Mastery. Each layer builds on the last, grounded in cognitive science research.

Bayesian confidence

Tracks what you know, not how many cards you flipped. Concepts resurface based on time-decay and your response history.

Concept maps

Hierarchical concept trees with summaries, course groups, and diagram support. See the big picture before drilling down.

Agent-friendly content

Plain Markdown + YAML in a folder. Any AI agent can generate it. ken only cares that IDs are unique.

Content and state separated

Course materials in ~/Documents/learn/subjects/. Progress in ~/.local/share/ken/. Your repo stays clean.
